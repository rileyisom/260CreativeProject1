Game Maker Studio 2

My development studio of preference is GameMaker Studio 2 by YoYo Games. GameMaker Studio 2 is made specifically for small gaming companies or solo developers like me, and is designed for the purpose of making fast running, smooth, and delightful video games. It is focused on 2D game development, but has some tools for making basic 3D games as well. It does not use mainstream programming languages like C++ or Java, but rather its own language, known as Game Maker Language (GML).
